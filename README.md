# Dribbler Bot
Extension built to improve players dribbling skills in pod games such as PHBA, RBA, HBA, EF, and so on. There is 3 .dlls folders in the ZIP. You must download the zip and extract it. Put the 3 .dlls in the Dependencies folder and then install the .exe.  If you need help using this extension please message me on discord Ahmed#8333
